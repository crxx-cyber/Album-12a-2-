# Album 12a

A simple web project built with vanilla HTML, CSS, and JavaScript.

## Project Structure

- `index.html` - Main HTML file
- `style.css` - Stylesheet for the project
- `script.js` - JavaScript code
- `README.md` - This file

## Getting Started

1. Open `index.html` in your web browser
2. Edit the HTML, CSS, and JavaScript files to build your project
3. Refresh the browser to see your changes

## Tips

- Use the browser's Developer Tools (F12) to debug your code
- Check the browser console for any JavaScript errors
- Test your responsive design by resizing the browser window
